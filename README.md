# Sistema de Inventario — Portero Seguro

Repositorio del **Sistema de Control de Activos y Trazabilidad** de Portero Seguro: una aplicación web
de inventario para instalaciones de seguridad electrónica (CCTV, control de accesos, redes y consumibles),
con guías de salida, trazabilidad por número de serie y kardex auditable.

## 📂 Dónde está la aplicación

La versión vigente vive en:

> **[`portero_seguro_v7/proyecto_mejorado/`](portero_seguro_v7/proyecto_mejorado/)**

Consulta su **[README](portero_seguro_v7/proyecto_mejorado/README.md)** para instalación, uso y despliegue,
y el **[CHANGELOG](portero_seguro_v7/proyecto_mejorado/CHANGELOG.md)** para el historial de versiones.

## 🚀 Arranque rápido

```bat
cd portero_seguro_v7\proyecto_mejorado
lanzar.bat
```

Requiere **Windows** y **Python 3.10+**. Levanta Flask + un proxy inverso Caddy y muestra la URL de acceso.

## 🧱 Stack

Python · Flask · SQLite · Jinja2 · ReportLab · Caddy (proxy inverso).
